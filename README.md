# Redux Counter App

A simple **Counter Application** built using **React** and **Redux (without Redux Toolkit)** to demonstrate core Redux state management concepts.

---

## Features

* Increment counter
* Decrement counter
* Reset counter
* Centralized state management using Redux

---

## Redux Concepts Used

* Store
* Action
* Reducer
* Dispatch
* Provider
* useSelector
* useDispatch

---

## Tech Stack

* React
* Redux
* React Redux

---

## Project Purpose

This project is created to understand **core Redux fundamentals** such as actions, reducers, and the Redux store without using Redux Toolkit.

---


## Conclusion

This counter app demonstrates how Redux works internally and helps build a strong foundation before moving to Redux Toolkit.
